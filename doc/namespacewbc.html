Enumeration Type Documentation

enum wbc::constraint_type

Two type of constraints are possible:

  • Cartesian constraints: The motion between two coordinate frames (root, tip) will be constrained. This can be used for operational space control, e.g. Cartesian force/position control, obstacle avoidance, ...
  • Joint constraints: The motion for the given joints will be constrained. This can be used for joint space control, e.g. avoiding the joint limits, maintaining a certain elbow position, joint position control, ...
